# 🚀 Wallet Tracker Bot (Web3)

Real-time blockchain monitoring bot that tracks wallet activity, detects swaps, and sends instant Telegram alerts.

---

## 🔥 Features

- 📡 Real-time wallet tracking (ETH balance changes)
- 🔁 Swap detection via DEX routers
- 🪙 Token in/out decoding (ERC20)
- 💸 Gas cost calculation
- 🤖 Instant Telegram alerts
- 🔕 Duplicate alert filtering (clean signals)
